Pathvisiojs Diagrammer

Create and Edit Diagrams Online

Install pathvisiojs on your site to create, view and edit diagrams in your browser. We use it for biological pathways, but you can use it for any type of diagram, from flowcharts to electrical circuits.

Shape Library

Including nodes of type rectangle, rounded rectangle, ellipse and pentagon and edges of type straight, segmented, elbowed and curved.

Semantic Annotation

Optionally add semantic annotations to graphical elements.

Customizable

Add your own PNG, JPG, GIF or SVG files as custom shapes. Add your own CSS file to specify custom styling, such as backgound color or font family.

Click the diagram to enable pan/zoom. Drag to pan or scroll in/out to zoom.
